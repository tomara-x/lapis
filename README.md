### building

- install rust: https://www.rust-lang.org/tools/install
- install egui dependencies: https://github.com/emilk/egui#demo
- on linux also install libjack-dev
- clone lapis
```
git clone https://github.com/tomara-x/lapis.git
```
- build it
```
cd lapis
cargo run --release
```
